What is it's purpose?

The purpose of making this website is to try and learn to use three types of css.

    The types are:
  1. Inline CSS
  2. Internal CSS
  3. External CSS

How to apply them?

You can apply inline css using style="" property
You can apply internal css using styles tag in head tag
And you can apply external css by linking a css file to the html file
